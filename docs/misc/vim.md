# Vim command guide

## Opening vim

```bash
vim file.txt
vim +23 file.txt # Starts vim at line 23 of the file
```

### Options

-----|-----
`-o`| Open multiple files in horizontally tiled windows
`-O` |Open multiple files in vertically tiled windows
`-p` | Open multiple files in seperate tabs
